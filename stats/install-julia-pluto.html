Install and Use Julia + Pluto

Step 1: Install Julia

Go to https://julialang.org/downloads and download the current stable release of Julia, using the correct version for your operating system (Linux x86, Mac, Windows, etc). Then run the installation as you normally would do for any other software program/application.

Step 2: Run Julia

After installing, make sure that you can run Julia. On some systems, this means searching for the ‘Julia X.X.X’ program installed on your computer (where X.X.X are numbers that refer to the current version of Julia that you downloaded); in others, it means running the command ‘julia’ in a terminal. Make sure that you can execute 1 + 1:

Step 3: Install Pluto

Next we will install Pluto, the notebook environment that we will be using during the course. Pluto is a Julia programming environment designed for interactivity and quick experiments.

Open the Julia REPL. This is the command-line interface to Julia, similar to the previous screenshot. Here you type Julia commands, and when you press ENTER, it runs, and you see the result.

To install Pluto, we want to run a package manager command. To switch from Julia mode to Pkg mode, type ] (closing square bracket) at the julia> prompt. It will look like the image below, where the Julia mode switches into the Pkg mode.

The line turns another color and the prompt changes to pkg>, telling you that you are now in package manager mode. This mode allows you to do operations on packages (also called libraries). To install Pluto, run the following (case sensitive) command to add (install) the package to your system by downloading it from the internet: ‘add Pluto’. You should only need to do this once for each installation of Julia:

This might take a couple of minutes to finish installing. Once this has finished, you will then exit out of the package manager by pressing the Delete key. This will bring you back to the REPL. At this point you will exit out of Julia by typing the command ‘exit()’ and pressing enter.

Now both Julia and Pluto are installed!

Running Julia + Pluto and Opening a Notebook

Repeat the following steps anytime that you need to run Pluto and work on a notebook.

Step 1: Start Pluto

Start the Julia REPL, like you did during the setup by either running the command ‘julia’ in a terminal or by finding and starting the Julia application/program. Once Julia has started, in the REPL, type: ‘import Pluto; Pluto.run()’. (Note that you don’t have to remember this command because Julia will remember your command history, just press the UP arrow key button and old commands, such as starting Pluto, will appear.)

This will open a browswer window located at http://localhost:1234/ followed by a unique identifying number after this that is generated randomly every time that you run Pluto. It should look something like the following:

If nothing happens in the browser the first time, close Julia and try again.

Step 2: Opening a Notebook From the Internet

From the main menu, you can create new notebooks, or open existing ones. Our assignments will always be based on a template notebook available from this GitHub repository and there will be links to each assignment on the Moodle class page. To start from an assignment template notebook, just paste the URL into the ‘Open a notebook’ box and press the Open button (or ENTER key).

When you launch Pluto for the second time, your recent notebooks will appear in the main menu. You can click on them to continue where you left off.

Step 3: Saving a Notebook

Always make sure that you save your notebook somewhere on your computer since what you have downloaded is only the assignment template. At the top of the page you will have the option to save the notebook at a specific location on your computer. Click on this box and navigate to a folder on your computer where you want to save the assignment and save it there.
